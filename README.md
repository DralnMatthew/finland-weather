## Requirements

This project requires:

- Java 8 SDK
- Python 3
- Scala
- Apache Spark
- Apache Kafka
- Apache Cassandra
- D3.js
- Node.js v14.18.1
- npm v6.14.15

## Usage

1. Get you own OpenWeatherMap Keyid and set it in docker-compose.yaml

2. Run `docker-compose up --build ` in Terminal.

Visit http://localhost:8081
